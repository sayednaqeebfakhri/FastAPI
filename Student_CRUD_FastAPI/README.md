A simple REST API built with FastAPI and SQLite to manage student records.

This project demonstrates full CRUD (Create, Read, Update, Delete) operations with clean API design and database integration.

🚀 Features

Create a new student record

Retrieve a single student by ID

Retrieve all students

Update student information

Delete student records

SQLite database integration (lightweight & fast)

Auto-generated API documentation (Swagger UI)


🛠️ Tech Stack

Python 🐍

FastAPI ⚡

SQLite 🗄️

Uvicorn (ASGI Server)


📁 Project Structure
